# KagiBox

パスワードマネージャー。データはすべてローカルに保存され、外部には送信されません。

## インストール

[Releases](https://github.com/Irony-s9c/kagibox/releases/latest) から `KagiBox_x.x.x_x64-setup.exe` をダウンロードして実行してください。

## 機能

- パスワードの暗号化保存（AES-256-GCM）
- パスワード生成
- サービスアイコン設定（PNG / JPG）
- 自動ロック・クリップボード自動クリア
- アプリ内自動アップデート
- 日本語 / English
